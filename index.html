Plugins and Keymappings

Plugins

WDF comes with a bunch of useful plugins.

General

  • ack.vim - Plugin for the Perl module / CLI script 'ack'
  • vim-powerline - The ultimate vim statusline utility.
  • vim-pasta - Pasting in Vim with indentation adjusted to destination context
  • ZoomWin - Zoom in/out of windows (toggle between one window and multi-window)

Navigation

Tree
  • bufexplorer.zip - Buffer Explorer / Browser
  • Command-T - Fast file navigation for VIM
  • LustyExplorer - Dynamic filesystem and buffer explorer
  • LustyJuggler - Switch very quickly between your active buffers
  • nerdtree - A tree explorer plugin for vim.
  • vim-rooter - Changes Vim working directory to project root (identified by presence of DVCS directory).
File
  • vim-easymotion - Vim motions on speed!

Editing

  • Align - Help folks to align text, eqns, declarations, tables, etc
  • AutoAlign - Automatically align as you type (autoalign): bib c c++ html maple matlab tex vim
  • vim-autoclose - This plugin for Vim enable an auto-close chars feature for you. For instance if you type an '(', autoclose will automatically insert a ')' and put the cursor between than.
  • matchit.zip - extended % matching for HTML, LaTeX, and many other languages
  • vim-pandoc - vim bundle for pandoc users
  • supertab - Perform all your vim insert mode completions with Tab
  • vim-surround - surround.vim: quoting/parenthesizing made simple
  • textile.vim - Textile for VIM
  • vim-unimpaired - unimpaired.vim: pairs of handy bracket mappings
  • YankRing.vim - Maintains a history of previous yanks, changes and deletes

Colorschemes

  • vim-colors-solarized - precision colorscheme for the vim text editor

Programming

General
  • vim-fugitive - fugitive.vim: a Git wrapper so awesome, it should be illegal
  • nerdcommenter - Vim plugin for intensely orgasmic commenting
  • vim-ragtag - ragtag.vim: ghetto HTML/XML mappings (formerly allml.vim)
  • tagbar - Vim plugin that displays tags in a window, ordered by class etc.
  • xmledit - A filetype plugin for VIM to help edit XML files
Php
  • vim-twig - Twig syntax highlighting, snipMate, etc.
Javascript
  • jQuery - Syntax file for jQuery
  • vim-coffee-script - CoffeeScript support for vim
  • vim-javascript - Vastly improved vim's javascript indentation.
  • vim-jst - A vim plugin for highlighting and indenting JST/EJS syntax
Ruby - Rails
  • vim-cucumber - Vim Cucumber runtime files
  • vim-endwise - endwise.vim: wisely add "end" in ruby, endfunction/endif/more in vim script, etc
  • vim-rails - rails.vim: Ruby on Rails power tools
  • vim-ruby-refactoring - Refactoring tool for Ruby in vim!
  • vim-rspec - Vim script to run the spec command
Html - Css
  • vim-haml - Vim runtime files for Haml, Sass, and SCSS
  • zencoding-vim - zen-coding for vim: http://code.google.com/p/zen-coding/

Keymappings

...

General

  • ,z - toggle window zoom

Navigation

Tree
  • ,be - open buffer explorer
  • ,bs - split open buffer explorer
  • ,bv - vertical split open buffer explorer
  • ,lf - opens filesystem explorer
  • ,lr - opens filesystem explorer at the directory of the current file
  • ,k - opens mini buffer explorer
  • ,lg - grep in currently open buffers
  • ,g - command bar shows names of your currently opened buffers in most-recently-used order
  • ,j - switch to your previously used buffer
  • ,n - toggle NERDTree
  • ,f - find the current file in NERDTree

Programming

General
  • ,t - toggle Tagbar window -

Custom

  • ,w - Fast saving
  • jj - Switch to normal mode like a boss
  • ,d - Close buffer
  • <C-k> - ->
  • ;; - append ; at end of line and save
  • <C-l> - =>
