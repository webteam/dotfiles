## Installation

1. `git clone git@github.com:webteam/dotfiles.git`
2. `cd dotfiles`
3. `rake install`

Your dotfiles are safe! WDF will not overwrite anything.


## Update

1. `cd to-your-dotfiles`
2. `git pull`
